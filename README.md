## ✨ Features

- 🔐 **Secure Admin Login** (admin / admin@123)
- 👤 **GitHub Profile Analysis** (avatar, bio, stats)
- 📊 **Top 10 Repos** (stars, forks, language)
- 📈 **Interactive Stats Chart** (Chart.js)
- 📱 **Fully Responsive** (mobile-first)
- 🎨 **Modern SaaS UI** (glassmorphism, gradients)

```bash

# Install

# Run

# Open
http://localhost:3000
```
